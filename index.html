Warstwy strony internetowej

Strony internetowe, podobnie jak cebula, czy ogry, maja warstwy. Czym sa te warstwy?

  • Warstwa 0 - Tresc
  • Warstwa 1 - HTML
  • Warstwa 2 - CSS
  • Warstwa 3 - JS
Warstwa 0 - Tresc

Warstwa 0 a wiec tresc to jest najwazniejszy element strony internetowej, ktory jest opakowany pozostalymi warstwami.

Warstwa 1 - HTML

Jest to warstwa, ktora opisuje warstwe tresci, jest to warstwa semantyczna, a wiec HTML. Warstwa ta nadaje sens tresci, a wiec opakowuje ja w znaczniki, ktore nadaja znaczenie tresci - jest to tzw. semantycznosc.

Warstwa 2 - CSS

To warstwa prezentacji - nadaje wyglad i styl powyzszym warstwom

Warstwa 3 - JS

Warstwa interakcji, a wiec JS, jest najtrudniejszym elementem frontendu.


Wazne: wyzsze warstwy wplywaja na warstwy nizsze. Np. CSS wplywa na zarowno HTML jak i tresc. JS wplywa na wszystkie elementy umieszczone nizej.

JavaSCript - wprowadzenie

ES6 = ECMASCript 2015 - ustandaryzowany przez orgnizacje ECMA skryptowy jezyk programowania. JS jak kazdy jezyk programowania ma swoja skladnie. Skladowe jezyka JS:

  • Wartosci - Wartosci dla JS sa tym co liczby dla matematyki.
  • Operacje porownywania
  • Literaly (literals)
  • Zmienne
  • Funkcje - skladaja sie z nazwy funkcji, nawiasow okraglych, oraz parametrow.
  • Obiekty

Skrypty dodajemy do strony na dwa sposoby:

  • bezposrednie pisanie skryptow w znaczniku 'script'
  • zalaczenie zewnetrznego pliku ze skryptami
